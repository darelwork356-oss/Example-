# Zenvio PWA

Esta es una Progressive Web App (PWA) para Zenvio, una plataforma de historias y relatos.

## Características PWA

- ✅ Service Worker implementado
- ✅ Manifiesto web configurado
- ✅ Caché offline
- ✅ Instalación en dispositivos
- ✅ Notificaciones push (preparado)
- ✅ Sincronización en segundo plano

## Instalación

1. Visita la aplicación en tu navegador
2. Busca el botón "Instalar App" que aparece automáticamente
3. O usa el menú del navegador para "Instalar Zenvio"

## Funcionalidades Offline

- Navegación básica
- Lectura de contenido cacheado
- Interfaz de usuario completa

## Tecnologías

- Service Worker con estrategias de caché avanzadas
- Web App Manifest
- Push Notifications API (preparado)
- Background Sync API (preparado)

## Compatibilidad

- Chrome/Edge: ✅ Completo
- Firefox: ✅ Completo
- Safari: ✅ Básico (sin instalación)
- Mobile: ✅ Completo
